Physical Chemistry
MOLE CONCEPT
  • Moles = given mass / molar mass
  • 1 mole = 6.022 × 10²³ particles (Avogadro's number)
  • Molarity M = moles of solute / volume in litres
  • Molality m = moles of solute / kg of solvent
  • % composition = (mass of element / molar mass) × 100

Organic Chemistry
GOC — GENERAL ORGANIC CHEMISTRY
  • Inductive Effect: −I (electron-withdrawing: halogens, NO₂, CN)  |  +I (electron-donating: alkyl groups)
  • Resonance stabilises carbocations and carbanions
  • Hyperconjugation: stabilises carbocations (more α-H = more stable)
  • Aromaticity (Hückel's rule): 4n+2 π electrons in planar ring
  • Electrophiles: electron-deficient (H⁺, NO₂⁺, X⁺, carbocations)
  • Nucleophiles: electron-rich (OH⁻, CN⁻, NH₃, halide ions)
  • SN1: carbocation intermediate → racemisation  |  reactivity: 3° > 2° > 1°
  • SN2: backside attack → inversion of configuration  |  reactivity: 1° > 2° > 3°
  • Markovnikov's Rule: H adds to C with more H (in electrophilic addition to alkenes)
  • Anti-Markovnikov (Peroxide effect): H adds to C with fewer H (free radical addition)

Inorganic Chemistry
CHEMICAL BONDING
  • VSEPR: shape depends on bond pairs + lone pairs on central atom
  • Bond order ↑ → bond length ↓, bond energy ↑
  • Hybridisation: sp (linear 180°), sp² (trigonal planar 120°), sp³ (tetrahedral 109.5°), sp³d (trigonal bipyramidal), sp³d² (octahedral 90°)
  • MOT: bond order = (Nb − Na)/2  |  Bond order > 0 = stable molecule
  • Dipole moment = 0 for symmetrical molecules (CO₂, BF₃, CCl₄) even if bonds are polar
COORDINATION COMPOUNDS
  • IUPAC: ligands listed alphabetically BEFORE the metal; anionic complex name ends in -ate
  • Oxidation state: balance charge from ligands + overall charge
  • CFT: Δ (crystal field splitting)  |  Strong field ligand = low spin = fewer unpaired electrons = diamagnetic
  • Weak field ligand = high spin = more unpaired electrons = paramagnetic
  • Spectrochemical series: I⁻ < Br⁻ < Cl⁻ < F⁻ < OH⁻ < en < CN⁻ < CO (left = weak, right = strong)
  • Magnetic moment: μ = √[n(n+2)] BM (n = unpaired electrons)
  • Colour of complexes: due to d-d transitions (absorption of visible light)
K₂Cr₂O₇ — POTASSIUM DICHROMATE
  • Preparation: chromite ore → Na₂CrO₄ → acidify → K₂Cr₂O₇
  • Colour: orange
  • Strong oxidising agent in acidic medium
  • Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
  • Oxidation state of Cr = +6
KMnO₄ — POTASSIUM PERMANGANATE
  • Preparation: MnO₂ → K₂MnO₄ (fusion) → electrolytic oxidation → KMnO₄
  • Colour: purple/violet
  • Oxidises in acidic, alkaline & neutral media
  • Acidic: MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
  • Oxidation state of Mn = +7
p-BLOCK, d-BLOCK & f-BLOCK
  • p-block: memorise oxidation states, oxides, hydrides of N, P, S, O, halogens
  • Anomalous properties of 2nd period: N (no d-orbital, max covalency 4), O (high electronegativity, H-bonding)
  • d-block: variable oxidation states due to incomplete d-subshell
  • Lanthanoid contraction: similar atomic radii → difficult separation
  • Actinoids: radioactive, complex oxidation states
Colour of complexes (d-d transition), magnetic behaviour, oxidation states of Cr/Mn, K₂Cr₂O₇ preparation

English

Easy rank booster · Practice 15–20 min daily · ~25 questions

Reading Comprehension + Grammar + Vocabulary
READING COMPREHENSION
  • Read passage FIRST, then questions
  • Answers are usually explicit in the text — don't assume
  • Look for synonyms/paraphrased answers in passage
  • Tone/inference questions: read carefully around key lines
  • Title questions: find the central theme, not a detail
VOCABULARY
  • Learn roots: bene (good), mal (bad), dict (say), port (carry), rupt (break), cred (believe)
  • Common prefixes: un-, dis-, im-, pre-, re-, mis-
  • Common suffixes: -tion, -ous, -ful, -less, -ment, -ity
  • Practice previous year synonym/antonym words
  • Context clues in fill-in-the-blank questions
COMMON GRAMMAR RULES
  • Subject-verb agreement: singular subject → singular verb; collective nouns (team, committee) → singular
  • Tense consistency: don't mix past and present in same sentence without reason
  • Articles: 'a' before consonant sound, 'an' before vowel sound, 'the' for specific/unique things
  • Prepositions for time: at (exact time), in (month/year/period), on (day/date)
  • Pronoun-antecedent agreement: pronoun must match noun in number and gender
  • Parallelism: items in a list must be in the same grammatical form
  • Misplaced modifiers: modifier should be closest to the word it modifies
  • Active → Passive: Object + was/were + past participle + by + subject
  • Direct → Indirect speech: change tense, pronouns, time expressions
FILL IN THE BLANKS — CLUE WORDS
  • "since / for" → present perfect or past perfect
  • "if ... will" → first conditional (possible future)
  • "if ... would" → second conditional (unreal present)
  • "if ... had ... would have" → third conditional (unreal past)
  • "neither ... nor", "either ... or" → verb agrees with nearest subject
ERROR DETECTION — CHECK ORDER
  • Step 1: Subject-verb agreement
  • Step 2: Tense error
  • Step 3: Wrong preposition or article
  • Step 4: Pronoun error
  • Step 5: Redundancy or wrong word choice
SENTENCE REARRANGEMENT (PARA-JUMBLE)
  • Find the OPENING sentence: introduces topic, no pronoun reference to unknown entity
  • Find MANDATORY PAIRS: pronoun → its antecedent
  • Find LOGICAL CONNECTORS: however, therefore, although, moreover, consequently
  • Find CONCLUSION: often contains 'thus', 'therefore', 'finally', 'in conclusion'
One unseen passage (5–6 Qs), 2–3 error detection, 2–3 fillers, 2–3 vocabulary questions — every year

02 · Data Structures
STACK (LIFO)
  • LIFO = Last In First Out
  • Push: stack.append(item)
  • Pop: stack.pop()
  • Peek/Top: stack[-1]
  • isEmpty: len(stack) == 0
  • Applications: undo operations, expression evaluation, recursion, parenthesis matching
QUEUE (FIFO)
  • FIFO = First In First Out
  • Enqueue: queue.append(item)
  • Dequeue: queue.pop(0)
  • Front: queue[0]
  • isEmpty: len(queue) == 0
  • Efficient: from collections import deque
  • Applications: scheduling, BFS, print queue

  • NOT NULL: column cannot have NULL values

DE MORGAN'S LAWS
  • (A·B)' = A' + B'  → NOT(AND) = OR of NOTs
  • (A+B)' = A'·B'  → NOT(OR) = AND of NOTs
  • NAND and NOR are called UNIVERSAL gates (any Boolean function can be built from them alone)
BOOLEAN LAWS
  • Identity: A+0 = A, A·1 = A
  • Null: A+1 = 1, A·0 = 0
  • Complement: A+A' = 1, A·A' = 0
  • Idempotent: A+A = A, A·A = A
  • Absorption: A+AB = A, A(A+B) = A

05 · Computer Networks
NETWORK DEVICES
  • Hub → broadcasts to ALL devices (dumb)
  • Switch → sends to specific device using MAC address
  • Router → connects different networks using IP address
  • Modem → modulation/demodulation (digital ↔ analog)
  • Gateway → protocol conversion between networks
  • Repeater → amplifies signal over long distances
  • Bridge → connects two similar networks (uses MAC)
NETWORK TOPOLOGIES
  • Bus: single cable, all devices connected to it
  • Star: all devices connected to central hub/switch
  • Ring: devices in a ring, token-based
  • Mesh: every device connected to every other
  • Tree: hierarchical star topology
  • Hybrid: combination of above

IP ADDRESSING
  • IPv4: 32-bit address (e.g., 192.168.1.1) — 4 octets
  • IPv6: 128-bit address — handles more devices
  • Class A: 0–127 (large networks)
  • Class B: 128–191 (medium networks)
  • Class C: 192–223 (small networks, most common)
  • Private IPs: 192.168.x.x, 10.x.x.x, 172.16.x.x
TRANSMISSION MEDIA
  • Twisted pair: cheap, common (LAN cables) — susceptible to interference
  • Coaxial: better shielding, used in cable TV
  • Optical fibre: fastest, light signals, immune to electromagnetic interference
  • Wireless: radio waves, microwaves, infrared
